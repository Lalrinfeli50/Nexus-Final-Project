# Software Company Web Application

This project is a basic full-stack web application designed to manage and display user comments. It uses Express.js for the backend, MongoDB for database management, and HTML, CSS, and JavaScript for the frontend. The application features a responsive design with smooth scrolling and various UI enhancements.

## Features

- **Express.js**: Backend server to handle API requests and serve static files.
- **MongoDB**: Database for storing user comments and data.
- **Responsive Design**: Optimized for various screen sizes with mobile navigation support.
- **Smooth Scrolling**: Enhanced user experience with smooth scrolling animations.
- **Preloader**: Loading animation displayed while the content is being loaded.

## Project Structure

- `index.html`: Main HTML file containing the structure of the webpage.
- `main.css`: CSS file for styling the webpage.
- `main.js`: JavaScript file for frontend functionalities and interactivity.
- `server.js`: Node.js file for backend server and API routes.

## Usage

- **Adding Comments**: Users can submit comments through the comment form on the webpage.
- **Viewing Comments**: Submitted comments are displayed on the webpage, fetched from the MongoDB database.
